Batch Processing

Batch processing troubleshooting steps:

For issues troubleshooting batch processing it's helpful to see the actual statuses of the BatchRunEntries and the BatchRuns. There are times when the entries get confused within the system or the shippers are doing an innaccurate workflow. The Below query is an example for Bold Logistics that helps you see the the entries the shippers have submitted into the server.

select Top 20 c.PlainText, Title, bp.CreatedBy, br.CreatedDate, bp.ID, * from BatchProcesses bp
        join BatchRuns br on bp.Id = br.BatchProcess_Id
        join Profiles p on p.Id = bp.Profile_Id
        join Clients c on c.id = p.Client_Id
        WHERE c.PlainText LIKE 'Bold%'
        Order by br.CreatedDate desc
        select Top 20 bp.id, c.PlainText, p.PlainText, bp.CreatedBy, Title,  bp.CreatedDate from BatchProcesses bp
        join BatchRuns br on bp.Id = br.BatchProcess_Id
        join Profiles p on p.Id = bp.Profile_Id
        join Clients c on c.id = p.Client_Id
        Order by bp.CreatedDate desc
