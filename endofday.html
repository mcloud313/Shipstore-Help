End Of Day

End of Day is a process ran at the profile level. This functionality is normally used to generate a manifest for a certain carrier.

End Of Day Already Running

If a client gets "End of Day Already Running" it's because the process has gotten stuck and the flag for their profile is stuck to 1. The field is called EODRunning in the database under the Profiles table. If we reset that back to 0 this will resolve the issue.
