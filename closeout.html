Closeouts

Closing out certain carriers is a functionality that can be found on the utility page of the ship section. This function allows the shipper to create a manifest of the shipments they've performed for said carrier since the last closeout. There have been issues getting these closeouts to work mainly with EasyPost. This is typically due to timeout errors.

No Manifest Output

We have run into the issue before where the client was using a web service for multiple carriers. When attempting to closeout they were not getting an error but were also not getting their manifest. This was due to the web service (EasyPost) being used for FedEx and USPS shipments.
